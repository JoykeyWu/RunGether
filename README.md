# RunGether 🏃

> 助盲跑伴侣 App —— 让每一位视障跑者都能自由奔跑

## 项目简介

RunGether 是一款专为助盲跑场景设计的 Android 应用。
视障跑者与陪跑员通过约1米牵引绳并排跑步，两台手机通过
蓝牙连接，陪跑员操作方向摇杆实时引导视障跑者，
遇到危险时可一键/摇一摇触发紧急求助。

## MVP 核心功能

- 🔵 **蓝牙配对连接** — 两端手机扫描配对，建立稳定通信
- 🎮 **方向摇杆引导** — 陪跑员滑动摇杆，盲人端语音播报+震动反馈
- 🚨 **紧急求助** — 摇一摇触发，自动拨打电话+发送GPS位置短信
- 📊 **跑步数据记录** — 实时记录时间/距离/配速，SQLite本地存储
- 📱 **双端差异化界面** — 盲人端极简大按钮，陪跑员端功能操控面板

## 技术栈

- **语言**：Kotlin
- **UI**：XML 布局
- **通信**：经典蓝牙 BluetoothSocket
- **语音**：Android TextToSpeech
- **定位**：Android GPS LocationManager
- **数据库**：SQLite
- **地图**：高德地图 SDK（后续版本）
